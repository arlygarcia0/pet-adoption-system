# pet-adoption-system

## Project Description
This project is a Java Object-Oriented Programming system that simulates a pet adoption center. Users can browse available pets (dogs and cats), view their breeds, and adopt pets from the system.

## Features
- View available dogs and cats
- Track breeds for each animal type
- Adopt pets from the system
- Manage pet inventory

## Animal Types
- Dogs
- Cats

## Breed Tracking
Each pet type includes breed information to represent different available animals in the shelter (e.g., Golden Retriever, Siamese, etc.).

## OOP Concepts Used
- Encapsulation
- Inheritance
- Polymorphism
- Abstraction

## Planned Classes and Objects
- Shelter
- User
- Pet (base class)
- Dog (inherits Pet)
- Cat (inherits Pet)
- Shelter / AdoptionCenter (manages system)
